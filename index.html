Overview

Perkenalkan, saya adalah seorang mahasiswa prodi Sistem Informasi. Saya memiliki minat pada Pemrograman Website dan Sains Data. Saya menyadari bahwa minat saya perlu diiringi dengan banyak belajar dan berlatih, untuk itu saya mencoba untuk tetap belajar baik melalui perkuliahan, bootcamp, YouTube, maupun artikel di internet.

Masih belum banyak projek yang saya buat. Hanya sebatas website sederhana seperti personal website ini yang coba saya buat menggunakan html, css, dan Javascript murni. Demikian halnya dengan sains data, saya baru dapat membuat projek sederhana.

Saya berharap dapat memiliki softskill dan hardskill yang dapat digunakan dalam sebuah projek sungguhan baik itu untuk keperluan penelitian maupun dalam dunia industri. Saya berharap bisa mendapat kesempatan untuk belajar secara lebih intensif sehingga jika pada akhirnya kesempatan (untuk mngemban projek sungguhan) itu datang setidaknya saya sudah berusaha untuk melakukan persiapan yang memadai.
